CSS Grid Layout

Specifications

  • 1fr (fraction) is a special unit of measurement introduced in this specification.
  • It is not measured in any specific units of measurement (px, em, rem, others).
  • It follows that we cannot use it along with the function calc().
  • This unit of measurement cannot be less than one, and also cannot take negative values.
  • It is calculated after all other values different from fr were calculated.

Specifications

  • auto — It behaves quite interestingly and uses a cunning algorithm to calculate the size.
  • In some situations, it may seem that this unit of measurement works just like fr.
  • The main difference, auto will be calculated before it is calculated fr.

Tracks

The container-network is of two types: the usual display:grid and the built-in display:inline-grid.

  • display: grid — generates the grid-container of the block level.
  • display: inline-grid — generates the grid string level.

Determination of a clear grid

The number of lines / columns is determined using the properties of grid-template-rows and grid-template-columns. Properties are not inherited.

  • grid-template-columns — sets the speakers.
  • grid-template-rows — sets columns.
  • grid-template-areas — sets cell identifiers.

Determination of an implicit grid

The grid-auto-flow property controls the automatic placement of the mesh elements without a clear position. The property is not inherited.

  • grid-auto-columns — sets the implicit speakers of the grid.
  • grid-auto-rows — sets implicit lines of the grid.
  • grid-auto-flow — sets cell identifiers.
